# pi-temp-relay
Basic temperature controlled relay on Raspberry PI that used temperatures from Victron Cerbo generic sensors.

Ruuvi Tag: https://ruuvi.com/ruuvitag/<br/>
Raspberry PI 3, 4, or 5
